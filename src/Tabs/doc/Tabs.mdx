# Tabs

The Tabs component consists of clickable tabs that are aligned side by side. The
`<TabNav/>` should describe the Tab navigation items and `<TabContents/>`
should describe the sections within. Any DOM node can be passed as children to
the tab section.

#### Import Syntax

```js
import Tabs, {
  TabNav,
  TabTitle,
  TabContents,
  TabSection
} from 'calcite-react/Tabs'
```

## Basic Usage

<Playground>
  {() => {    
      class TabStory extends React.Component {
        constructor(props) {
          super(props);
          this.state = {
            activeTabKey: 0
          };

          this.onTabChange = this.onTabChange.bind(this);
        }

        onTabChange(activeTabKey) {
          this.setState({ activeTabKey });
        };

        render() {
          return (
            <Tabs
              onTabChange={this.onTabChange}
              activeTabKey={this.state.activeTabKey}
            >
              <TabSection title="Tab 1">Tab 1 content</TabSection>
              <TabSection title="Tab 2">Tab 2 content</TabSection>
              <TabSection title="Tab 3">Tab 3 content</TabSection>
            </Tabs>
          );
        }
      }

      return <TabStory />;
    }}

</Playground>
