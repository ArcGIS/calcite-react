# Alert

Alerts are used to inform users of state changes, errors, or successful actions.

## Basic Usage

    </Alert>
  </div>
  <div label="red">
    <Alert red>Something Happened!</Alert>
  </div>

    <Alert yellow>Something Happened!</Alert>
  </div>
  <div label="green">
    <Alert green>Something Happened!</Alert>
  </div>
  <div label="full">
    <Alert full>Something Happened!</Alert>
  </div>
