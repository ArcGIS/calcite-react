# Tooltip

Tooltip allows users to add an element at a target position when the user hovers
over an element.

## Basic Usage

<Playground>
  <Tooltip title="Basic Tooltip">
    <CalciteA>Hover Me</CalciteA>
  </Tooltip>
</Playground>
