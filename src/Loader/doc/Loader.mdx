# Loader

The Loader element is a placeholder while content is being retrieved or
rendered.

#### Import Syntax

```js
import Loader from 'calcite-react/Loader'
```

## Basic Usage

<Playground>
  <Loader />
</Playground>

## With Text

<Playground>
  <Loader text="Loading..." />
</Playground>

## Custom Size

<Playground>
  <Loader sizeRatio={2} />
  <Loader sizeRatio={0.4} />
</Playground>

## Custom Color

<Playground>
  <Loader color="tomato" />
  <Loader color="#1E90FF" />
  <Loader color={EsriColors.Calcite_Gray_300} />
</Playground>
