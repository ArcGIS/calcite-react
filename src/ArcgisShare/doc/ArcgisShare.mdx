# ArcgisShare

A checkbox tree to select which groups you want to share an item with in ArcGIS.

#### Import Syntax

```js
import ArcgisShare from 'calcite-react/ArcgisShare'
```

## Basic Usage

<Playground>
  <ArcgisShare user={user} portal={portal} onChange={e => console.log(e)} />
</Playground>

## Populated with an ArcGIS Item

<Playground>
  <ArcgisShare
    user={user}
    portal={portal}
    sharing={item.sharing}
    onChange={e => console.log(e)}
  />
</Playground>

## Promote Favorites

<Playground>
  <ArcgisShare
    user={user}
    portal={portal}
    promoteFavorites
    onChange={e => console.log(e)}
  />
</Playground>

## No Groups

<Playground>
  <ArcgisShare
    user={itemNoGroups}

    promoteFavorites
    onChange={e => console.log(e)}
  />
</Playground>
