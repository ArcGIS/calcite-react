# Panel

Panels are useful to draw attention or contain certain text on a page. They are
simple containers, which can be used to help guide users through the visual
hierarchy of your design.

## Basic Usage

<Playground>
  <Panel>
    <PanelTitle>This is a panel.</PanelTitle>
    <PanelText>
      Panels set <code>background-color</code> and frame content.
    </PanelText>
  </Panel>
</Playground>
