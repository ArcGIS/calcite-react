# SubNav

SubNav provides a second level of hierarchy to a site. While TopNav includes the
top-most level of navigation, SubNav includes navigation for sections nested
inside that. SubNav has a tabbed navigation pattern and a title. 

### ⛔️ Use of `legacy` prop ⛔️

The `legacy` prop must be included to keep original styles from before the `v0.55.0` release. 
`SubNavActions` may only be used with legacy. Please refer to the `SubNav` props 
table for a full list of props that are only accepted with `legacy`.

#### Import Syntax

```js
import SubNav, {
  SubNavTitle,
  SubNavList,
  SubNavLink,
  SubNavActions
} from 'calcite-react/SubNav';
```
