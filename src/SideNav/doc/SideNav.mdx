# SideNav

SideNav can be used either for in-page navigation, or for lists of articles
under a given topic.

#### Import Syntax

```js
import SideNav, {
  SideNavTitle,
  SideNavLink
} from 'calcite-react/SideNav'
```

## Basic Usage

<Playground>
  <SideNav>
    <SideNavTitle>Sidenav Group</SideNavTitle>

      <SideNavLink>Transition</SideNavLink>
      <SideNavLink>Perpetual</SideNavLink>

    </nav>
  </SideNav>
